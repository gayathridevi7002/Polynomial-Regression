Polynomial Regression

Fitting curves to data — beyond the straight line

Section 01

Concept Description

Definition

Polynomial Regression is a supervised machine learning algorithm that models the relationship between an independent variable x and a dependent variable y as an nth-degree polynomial. It extends simple linear regression by allowing the regression curve to be a polynomial rather than a straight line, enabling it to capture non-linear patterns in data.

y = β₀ + β₁x + β₂x² + β₃x³ + ... + βₙxⁿ + ε Where: β₀, β₁, ..., βₙ → coefficients (learned from data) n → degree of the polynomial ε → error term (noise)

Purpose

Real-world data rarely follows a perfectly straight line. Polynomial regression allows us to fit curved models that better represent the underlying pattern — like how temperature varies with time of day, how a car's braking distance increases with speed, or how a plant grows over weeks.

Non-linear modelling Curve fitting Predictive analytics Scientific modelling

Step-by-Step Explanation

1
Collect DataGather pairs of (x, y) observations. For example, hours studied vs exam score.
2
Choose Degree (n)Decide the polynomial degree. Degree 1 = straight line; Degree 2 = parabola; higher degrees = more complex curves.
3
Feature TransformationTransform x into a feature matrix: X = [1, x, x², x³, …, xⁿ]. This converts the problem into a linear regression in the transformed feature space.
4
Set Up Normal EquationsUse the Ordinary Least Squares (OLS) method: compute the design matrix X and find coefficients β = (XᵀX)⁻¹ Xᵀy.
5
Solve for CoefficientsSolve the normal equations to get the best-fit coefficients β₀, β₁, …, βₙ that minimise the Sum of Squared Errors (SSE).
6
Predict & EvaluateUse the fitted polynomial to predict ŷ for new x values. Evaluate using R², MSE, or RMSE.

Important Parameters

ParameterSymbolDescription
DegreenControls complexity of the curve. Higher = more flexible but risks overfitting.
Coefficientsβ₀…βₙWeights learned by the model via OLS.
Design MatrixXMatrix of polynomial features [1, x, x², …, xⁿ].
MSE—Mean Squared Error — average of squared differences between actual and predicted.
R² ScoreR²Proportion of variance explained by the model. Closer to 1 is better.
Overfitting—When degree is too high, model fits noise rather than the true pattern.

Step-by-Step Worked Example

Let's walk through polynomial regression by hand using a small dataset with Degree = 2. Each step below shows exactly what the algorithm does — from raw data to final predictions.

🎯 Goal
Find the best-fit curve ŷ = β₀ + β₁x + β₂x² through 5 data points, so predictions are as close to actual y-values as possible.

💡 Why add 1s? The constant column lets the formula have an intercept (β₀). Without it, the curve would always pass through the origin.
STEP 2

Compute Xᵀ X and Xᵀ y

We transpose X and multiply to "summarise" the data into a compact 3×3 system. This is the core of the Ordinary Least Squares (OLS) method.

XᵀX  (3×3 matrix)
| 5 15 55 | | 15 55 225 | | 55 225 979 |
Each cell is the dot product of two columns of X
Xᵀy  (3×1 vector)
| 60 | | 228 | | 900 |
Each value = column of X dotted with y
🔢 Verification example: XᵀX[0][0] = 1²+1²+1²+1²+1² = 5  ·  XᵀX[1][1] = 1²+2²+3²+4²+5² = 55
STEP 3

Solve β = (Xᵀ X)⁻¹ Xᵀ y

We invert XᵀX and multiply by Xᵀy using Gaussian elimination. This gives us the 3 coefficients that minimise the total squared error.
